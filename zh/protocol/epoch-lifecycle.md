# 轮次生命周期

Surface Finance 金库以称为**轮次**的周期运行。每个轮次是一个完整的策略循环——从开放存款、部署资本，到返还收益。了解各阶段可帮助您知晓何时可以存款、何时资金在运作，以及何时可以提取。

## 各阶段概览

| 阶段 | 我可以存款吗？ | 我可以提取吗？ |
|---|---|---|
| 募集期 | ✓ 是 | ✓ 是 |
| 交易期 | ✗ 否 | ✗ 否 |
| 提取开放期 | ✗ 否 | ✓ 是 |

---

## 第一阶段 — 募集期

金库开放具有固定起止时间的**募集窗口**。在此窗口期间：

- 您可以存入代币并获取金库份额
- 若您已持有前一轮的份额，也可以自由提取
- 金库尚未部署资本——目前没有任何资产处于风险中

可能存在存款上限或访问限制。若您的存款被拒绝，可能是该轮次金库已满容量。

---

## 第二阶段 — 交易期

募集窗口关闭后，金库进入**活跃交易阶段**。策略部署资本，利用底层代币的波动性进行操作。

在此阶段：
- **所有用户的存款和提取均暂停**
- 您的份额仍代表您对金库的完整按比例所有权
- 策略持续运行直至结束——您无需采取任何行动

### 交易持续多长时间？

每个轮次均有预定的结束日期。根据策略不同，在预定结束日期后资金完全结算并返还金库可能需要一段短暂时间：

- 资金通常在轮次结束日期后的 **72 小时内**返还
- 某些情况下，若市场条件需要，持仓可能在轮次结束后最多保持 **30 天**

提取将一直锁定至资金完全返还金库为止。

---

## 第三阶段 — 提取开放期

策略结束并将资金返还金库后，提取窗口开放。金库的每份额价格将更新以反映策略的净结果——盈利或亏损。

此时您可以：
- **提取** — 按更新后的代币价值赎回份额
- **滚续** — 不采取任何行动，您的份额将自动进入下一轮次的募集窗口

{% hint style="info" %}
无需采取任何行动即可滚续。若您将份额保留在金库中，它们将自动参与下一轮次。
{% endhint %}

{% hint style="warning" %}
若策略发生亏损，金库的每份额价格将低于您存入时的价格。所有持份人将按比例分担此亏损——协议层面不提供任何亏损保护。
{% endhint %}
