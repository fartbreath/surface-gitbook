# 金库运作原理

Surface Finance 金库是将您的 Meme 币资产付诸实践的链上基础设施。每个金库基于 ERC-4626 标准构建——这是广泛采用的收益型金库规范——将您的存款与机构级交易策略相连接，所有核算均在链上透明处理。本页介绍您的代币在金库中的去向以及持仓的追踪方式。

## 您的持仓：金库份额

当您向 Surface Finance 金库存入代币时，您将获得**金库份额**——一种代表您对金库按比例所有权的 ERC-20 代币。

每份份额的价值由金库的**净资产价值（NAV）**除以流通总份额数决定：

$$\text{份额价格} = \frac{NAV}{\text{总份额数}}$$

随着策略产生收益，NAV 增加，每份份额可兑换的代币便多于存入时。若策略发生亏损，情况相反——每份份额价值低于存入时。

**存入时**，金库按当前份额价格铸造份额：

$$\text{获得的份额数} = \frac{\text{您的存款}}{\text{份额价格}}$$

**提取时**，金库销毁您的份额并返还等值代币：

$$\text{返还的代币数} = \text{销毁的份额数} \times \text{份额价格}$$

所有份额核算均在链上处理，可公开核查。

## 交易期间您的代币去哪了

当金库进入活跃交易阶段，策略将托管金库资金以执行交易。在此窗口期间：

- 金库的链上代币余额将为零或接近零——这是正常现象
- 您的份额仍代表您对金库的按比例所有权
- 金库追踪部署到策略中的全部金额，因此托管转移本身不影响您的份额价值
- 在策略结束并返还资金之前，您无法存入或提取

策略结束后，净收益（盈利或亏损）将返还至金库合约，提取窗口重新开放。

{% hint style="info" %}
该策略旨在从市场波动中产生收益——而非预测价格方向。这意味着无论价格上涨还是下跌，您的代币都在积极创造价值。
{% endhint %}

## 访问权限与存款限制

部分金库可能设有：
- **最大存款上限** — 一旦达到上限，新存款将不被接受，直至下一轮次开始
- **白名单要求** — 早期阶段可能仅限批准地址访问

{% hint style="warning" %}
存款和提取仅在其指定的窗口期间可用。在活跃交易阶段，**所有存款和提取均暂停**。
{% endhint %}
