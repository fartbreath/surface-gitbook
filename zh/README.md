# Surface Finance

Surface Finance 将机构级收益策略带入 Meme 币领域——无论市场上涨、下跌还是横盘，都能让您闲置的代币持续创造收益。

## 什么是 Surface Finance？

Meme 币的本质是波动性。大多数持有者要么投机价格，要么坐等下一波行情。Surface Finance 提供了第三种选择：将您的代币存入金库，让主动管理策略从这种波动性本身中获取收益。

该策略采用完全透明的账务核算。您存入代币，换取金库份额；当策略产生收益时，每份份额将兑换更多代币。当您想退出时，按当前价值赎回份额——除活跃交易窗口外，不存在锁仓期。

{% hint style="info" %}
波动性是加密世界中唯一不变的常数——无论牛市还是熊市，它从未消失。Surface Finance 正是为将这种常数转化为收益而生。
{% endhint %}

## 运作方式

1. **存入** — 当金库开放存款时，发送您的代币并按当前价格获取金库份额。
2. **策略运行** — 金库将资本部署到交易策略中。此期间，存入和提取均暂停。
3. **提取** — 策略结束后，金库重新开放。按更新后的价值提取，或保留份额滚入下一轮。

## 核心属性

| | |
|---|---|
| 存入代币 | 每个金库接受单一 ERC-20 代币 |
| 您的持仓 | 金库份额——可转让，兼容 ERC-20 |
| 策略节奏 | 基于轮次（募集 → 交易 → 提取） |
| 访问权限 | 您的份额归您所有——存于您的钱包，可在提取窗口兑换 |
