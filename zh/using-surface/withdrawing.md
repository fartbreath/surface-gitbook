# 提取

本页说明如何以及何时可以从 Surface Finance 金库提取资金。

## 何时可以提取？

提取在以下两个窗口期间可用：

| 金库阶段 | 提取情况 |
|---|---|
| 空闲期 | ✓ 开放 |
| 已排期（募集开始前） | ✓ 开放 |
| 募集期 | ✓ 开放 |
| 交易期（资金在金库内） | ✗ 锁定 |
| 交易期（资金已托管） | ✗ 锁定 |
| 平仓期 | ✗ 锁定 |
| 提取开放期（轮次后） | ✓ 开放 |

{% hint style="warning" %}
当金库处于活跃轮次或资金由交易员托管期间，提取将**锁定**。状态选项卡中的**"提取已锁定"**标识表示此状态。
{% endhint %}

---

## 提取资产 vs. 赎回份额

金库支持两种退出持仓的方式：

**按资产金额提取** — 指定您希望收到的底层代币数量。金库将计算并销毁相应份额。

$$\text{销毁份额数} = \frac{\text{请求资产数} \times S}{NAV}$$

**按份额数量赎回** — 指定要销毁的金库份额数量。金库将计算并返还相应底层代币。

$$\text{返还资产数} = \frac{\text{销毁份额数} \times NAV}{S}$$

两种方式结果相同——选择更方便的即可。

## 提取步骤

1. 连接钱包并在金库管理前端加载金库合约地址
2. 前往**存款人**选项卡
3. 确认状态选项卡中已显示**"提取开放"**标识
4. 在**"提取资产"**下，输入资产金额并点击**提取**\
   — 或在**"赎回份额"**下，输入份额数量并点击**赎回**
5. 在钱包中确认交易
6. 确认后，您的代币将出现在您的钱包中

---

## 亏损情况下提取

若策略在该轮次中发生亏损，每份额的 NAV 将低于存入时。在亏损轮次后提取意味着您收到的底层代币少于存入时——亏损按所有持份人的比例分摊。金库合约在协议层面不提供任何亏损保护。

---

## 滚续到下一轮次

若轮次结束后您未进行提取，您的份额将自动滚续到下一轮次。当运营方再次调用 `startEpoch` 时，金库重新进入募集阶段，您的份额继续参与。

无需任何操作即可滚续——只需将份额留在金库中即可。

---

## 紧急情况 / 暂停状态

若金库被运营方暂停，所有提取将暂时被阻止。这是应对紧急情况的安全机制。请监控金库事件（`Paused`、`Unpaused`）或金库状态仪表板以了解状态变化。
