# 安全与透明度

将机构级策略带上链意味着坚守机构所期望的同等标准：透明的核算、经过审计的合约以及明确定义的规则。本页介绍作为存款人您应了解的关键属性。

## 您的份额与您的资本

理解以下两点的区别非常重要：

**您的金库份额**存储在您自己的钱包中。Surface Finance 从不代您持有份额，只有您才能赎回或转让这些份额。

**您的存款资本**在活跃交易阶段由 Surface Finance 的专业交易团队管理。这是产品的核心——机构级团队主动管理您的代币，从市场条件中获取收益。金库合约执行关于何时以及如何转移资金的规则，所有资金流动均记录在链上。

可以将其理解为管理基金的链上等价物：您的持仓归您所有，您的资本在创造价值。

## 开源合约

Surface Finance 金库合约是开源的，可公开验证。金库基于 ERC-4626 标准构建，使用经过审计的 OpenZeppelin 组件，包括：

- **ERC-4626** — 标准化的链上金库接口
- **ReentrancyGuard** — 防止所有资金转移函数遭受重入攻击
- **SafeERC20** — 确保所有 ERC-20 代币的安全转账行为

## 链上核查

所有金库活动均记录在链上，可公开读取：

| 您可以查看的内容 | 查看位置 |
|---|---|
| 当前金库余额（总资产） | 金库合约上的 `totalAssets()` |
| 您的份额余额 | 您钱包中金库代币的 ERC-20 余额 |
| 您份额的当前价值 | `convertToAssets(yourShares)` |
| 存款是否开放 | `isFunding()` |
| 轮次是否活跃 | `isInEpoch()` |
| 当前轮次时间信息 | `getCurrentEpochInfo()` |

## 风险披露

- **对手方风险** — 在交易阶段，资本由交易团队部署。收益取决于该团队的表现和诚信。在存入大量资本之前，请了解 Surface Finance 的历史记录和团队。
- **策略风险** — 策略返还的金额可能少于存入金额。亏损将按比例分摊给所有持份人。
- **智能合约风险** — 尽管经过审计并使用成熟的代码库，智能合约仍可能存在漏洞。请仅存入您能承受损失的资金。
- **流动性风险** — 在活跃交易阶段，提取暂停。您将无法在轮次结束之前退出。
